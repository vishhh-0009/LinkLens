# LinkLens – Smart URL Analytics Platform

A full-stack URL shortening and analytics platform built with React, Node.js, Express, and MySQL.

## Features

- URL Shortening
- Custom Aliases
- QR Code Generation
- Click Tracking
- Expiry Date Management
- Analytics Dashboard
- Top Performing Links
- Activity Feed
- Search & Filters
- Responsive UI

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- Chart.js

### Backend

- Node.js
- Express.js

### Database

- MySQL

## Installation

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
